# mirrorkit-site

Landing page, privacy policy and support page for [MirrorKit](https://github.com/TrabelsiAchraf/MirrorApp) — a native macOS app that mirrors your iPhone or iPad on your Mac over USB.

🌐 **Live site**: https://trabelsiachraf.github.io/mirrorkit-site/

## Contents

- `index.html` — marketing landing page
- `support.html` — support / contact page (required by the App Store)
- `privacy.html` — privacy policy (required by the App Store)
- `hero.png`, `shot-*.png` — screenshots used on the landing page

## Hosting

Served via GitHub Pages from the `main` branch. Any push to `main` is published within a minute.

## Local preview

```bash
python3 -m http.server 8000
# then open http://localhost:8000
```
